Bienvenue dans l'application OCT-Classifier.

Cette application a pour but de classifier une image d'OCT afin de déterminer si un patient a ou non une maladie des yeux. Les classes possibles sont :

  • Normal
  • CNV : la dégénérescence maculaire liée à l'âge
  • DRUSEN : la dégénérescence maculaire liée à l'âge présentant des drusen cuticulaires et des pseudodrusen réticulaires
  • DME : l'œdème maculaire lié au diabète

Pour utiliser cette application, cliquez sur le bouton charger une image afin de charger une image obtenue par imagerie OCT puis appuyez sur le bouton classifier pour obtenir un résultat.
